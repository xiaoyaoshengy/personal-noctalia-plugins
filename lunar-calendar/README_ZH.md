# 农历插件

\[[English](./README.md)\][中文]

用于[Noctalia-Shell](https://github.com/noctalia-dev/noctalia-shell)的农历插件

## 功能

- **顶栏工具**：展示当前日期的农历信息
- **面板**：仿Noctalia-Shell日历的农历面板，增加公历日期下的农历信息
- 支持节日

## 安装方式

- 在`设置 >> 插件 >> 来源`中添加`https://github.com/xiaoyaoshengy/personal-noctalia-plugins`，刷新插件列表，安装`lunar-calendar`插件
- 或者克隆此文件夹到`~/.config/noctalia/plugins`
